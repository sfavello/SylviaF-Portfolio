Weather Api App

This is a fully functional Weather App using an APIKey from OpenWeatherMap.org. I used, Javascript, CSS, & HTML for this application. Since this was my first time using APIkeys into my own code, that came with many challenges. Luckily I was able to overcome them, learning how to apply for an APIKey, read the documentation for the APIKey, figure out which data I wanted and didn't want, customization, and how to revolve my code around that information. I also learned how to better stylize with CSS. I was able to add opacity to my section, create a type input search bar and align it with my button. I learned how to sync items together and make a cohesive design alltogether. Overall, I learned how to start from beginning to end how to create a fully functional application with the user in mind.
